# ローカルAI文字起こし

**プライバシー重視のローカル会議文字起こし — すべての処理はMac上で完結**

Discord、LINE、Zoom、Teams、FaceTime — Macから出るすべての音を録音して、リアルタイムで文字起こし。相手の声もあなたの声も、まるごとキャプチャ。通話アプリを問わず、Macの画面から出る音ならなんでも録音＆文字起こしできます。

## 特徴

- **あらゆる通話アプリに対応** — Macのシステム音声をキャプチャするため、どんな通話アプリでも録音可能。アプリごとの対応は不要
- **2つの文字起こしエンジン** — Apple音声認識によるリアルタイム文字起こし + Whisper AIによる高精度な録音後文字起こし
- **外国語もリアルタイム翻訳** — 翻訳もすべてデバイス上で処理
- **カレンダー連携** — Zoom・Meet・Teams等のビデオ会議予定を検知し、開始前に録音開始を提案する通知を送信。対象カレンダーの選択も可能
- **無音自動停止** — 会議終了後に一定時間音声が検出されなければ自動で録音を停止。録音の止め忘れを防止
- **メニューバー対応** — メニューバーからすばやく録音の開始・停止が可能。録音中は経過時間を表示
- **プライバシーファースト** — 音声データも文字起こし結果も翻訳も、一切サーバーに送信しません

## インストール

Mac App Store からダウンロードできます（macOS 26 Tahoe 以降）。

## こんな使い方に

- オンライン会議の記録: Zoom・Meet・Teamsの会議を丸ごと文字起こし
- Discord・LINE通話: ゲームやプライベートの通話も録音
- 海外との会議: 英語の発言をリアルタイム翻訳
- 講義やセミナー: リアルタイムで内容を追える
- お好みのAIに文字起こしを渡して議事録やToDoを生成
